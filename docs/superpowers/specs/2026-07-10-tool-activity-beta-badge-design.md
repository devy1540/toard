# AI 도구 활동 베타 태그 디자인

## 목적

개요 화면의 `AI 도구 활동` 카드가 베타 기능임을 기존 대시보드의 상태 표현과 일관되게 알린다.

## 범위

- `AI 도구 활동` 카드 제목 바로 오른쪽에 `FeatureStatusBadge`를 배치한다.
- 상태는 `beta`를 사용해 기존 민트색 베타 스타일을 재사용한다.
- 문구는 기존 `nav.badge.beta` 번역을 사용해 한국어에서는 `베타`, 영어에서는 `beta`로 표시한다.
- 카드는 현재처럼 개요 화면에만 렌더링한다.
- 도구 상세 페이지, 클래식 화면, 사이드바에는 새 태그를 추가하지 않는다.

## 구현

`ToolActivityCard`의 제목 영역을 가로 정렬하고 제목과 배지 사이에 작은 간격을 둔다. 별도 배지 컴포넌트나 카드 전용 상태 API는 만들지 않는다.

## 검증

- 소스 회귀 테스트가 `FeatureStatusBadge`의 `beta` 사용과 기존 번역 키 사용을 확인한다.
- web 테스트와 TypeScript 타입 검사를 통과해야 한다.
- 로컬 개요 화면에서 제목 옆에 베타 태그가 보이고 클래식 화면에는 카드가 렌더링되지 않는 것을 확인한다.
