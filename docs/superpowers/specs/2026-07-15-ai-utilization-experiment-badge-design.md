# AI 활용 지수 실험 태그 디자인

## 목적

AI 활용 지수가 확정된 평가 기준이 아니라 검증 중인 방법론임을 개인과 조직 화면에서 명확히 알린다.

## 표시 범위

- 개인 인사이트의 `AI 활용 지수` 카드 제목 옆에 태그를 표시한다.
- 조직 현황의 `조직 AI 활용 지수` 카드 제목 옆에 같은 태그를 표시한다.
- 내비게이션, 페이지 제목, 기존 인사이트 Beta 표시는 변경하지 않는다.

## 표현

- 기존 공용 `Badge` 컴포넌트의 `outline` variant를 사용한다.
- 한국어 문구는 `실험`, 영어 문구는 `Experimental`이다.
- 개인 카드의 신뢰도 배지는 기존 우측 위치와 의미를 유지한다.
- 실험 태그는 제목과 같은 줄에 배치하되 좁은 화면에서는 자연스럽게 줄바꿈할 수 있다.

## 데이터와 동작

태그는 정적 제품 상태 표식이다. 활용 지수 산식, 캐시, 저장소 조회, 개인정보 보호 기준과 조직 최소 표본 정책은 변경하지 않는다.

## 검증

- 개인·조직 카드가 각각 번역된 실험 태그를 렌더하는지 source contract test로 고정한다.
- 한국어·영어 메시지 구조가 동일한지 확인한다.
- 웹 타입 검사와 전체 웹 테스트를 실행한다.
