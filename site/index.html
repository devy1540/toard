수집·비용·조직 뷰까지 — Collector 없이 앱 하나로.

멀티 프로바이더

Claude Code · Codex 등 OTLP를 내보내는 도구를 하나의 대시보드로 수렴.

경량 수집

Collector·별도 파이프라인 없이 앱이 OTLP/JSON을 직접 수신. 멱등 dedup 내장.

정확한 비용

LiteLLM 가격 기반 비용 엔진 — tiered·캐시·fast 요금까지, 일 단위 자동 동기화.

조직 뷰

조직·팀 집계 · 리더보드 · 개인 대시보드 · 관리자 패널 · 초대 기반 온보딩.

확장 가능한 저장소

기본은 Postgres 단일, 중규모 이상은 ClickHouse 옵트인.

유연한 인증

OAuth(GitHub/Google) · id/pw · open 모드를 조직 환경에 맞게 선택.

셀프호스팅

Docker Compose 한 줄부터 Kubernetes/Helm 무중단 배포까지.

타임존 지원

ORG_TIMEZONE 기준 일별 집계 — 어느 조직 시간대든 "하루"가 정확.

동작 방식

shim이 CLI를 투명하게 래핑 → OTLP 전송 → 정규화·비용 계산 → 대시보드.

claude / codex toard shim toard 대시보드

한 줄로 시작

Docker Compose로 띄우고, CLI로 붙이면 끝.

bashtoard up
$ docker compose up -d
$ npx @toard/cli up
# → http://localhost:3000 에서 대시보드 확인
